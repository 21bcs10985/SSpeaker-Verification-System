About Project

This project is a speaker verification system that utilizes deep learning techniques to authenticate users based on their voice recordings. It incorporates a deepfake detection model to differentiate between real and synthetic voices, ensuring the authenticity of the speaker. The system processes the recorded audio, extracts relevant features, and uses a trained DNN for speaker recognition. Additionally, the project provides a user-friendly interface for recording, saving, and analyzing voice data, making it suitable for secure voice authentication in various applications.
